# Welcome to MD Viewer!

This is a **live preview** of your markdown.

## Features
- Split view editing
- Real-time rendering
- Dark Mode support
- **Local file loading**
- **Mermaid Diagrams**

### Code Example
```javascript
function helloWorld() {
  console.log("Hello, World!");
}
```

### Flowchart Example
```mermaid
graph TD
    A[Start] --> B{Is it working?}
    B -->|Yes| C[Great!]
    B -->|No| D[Debug]
    D --> B
```

### Sequence Diagram (mmd)
```mmd
sequenceDiagram
    Alice->>John: Hello John, how are you?
    John-->>Alice: Great!
    Alice-)John: See you later!
```
